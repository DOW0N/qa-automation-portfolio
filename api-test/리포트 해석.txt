📌 Newman HTML Report 해석

Newman HTML 리포트는 API 자동화 테스트 실행 결과를 시각적으로 확인하기 위해 사용하였습니다.
단순 성공/실패 여부가 아니라, 응답 시간, 테스트 커버리지, 실패 원인을 한 눈에 파악할 수 있어
릴리즈 전 품질 판단 자료로 활용 가능합니다.

📌 실패 케이스 테스트 해석

잘못된 로그인 요청 시 서버가 401 상태 코드와 오류 메시지를 정상적으로 반환하는지 검증하였다.
이를 통해 서버의 예외 처리 로직이 정상 동작함을 확인하였다.

📌 보안 관점 테스트 해석 (중요)

인증 토큰 없이 사용자 장바구니 API 접근이 가능한 동작을 확인하였다.
기능 테스트 관점에서는 현재 API 스펙에 따라 테스트를 통과 처리하였으나,
QA 관점에서는 보안 리스크로 판단하여 리포트에 명시하였다.
